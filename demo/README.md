admin-ui-plugin-js DEMO
=======================

## Installation
+ Git clone the project to your computer:
```
$ git clone git@github.com:triniti/admin-ui-plugin-js.git
```
+ Open terminal and `cd admin-ui-plugin-js/demo`
+ Run `$ npm install`
+ Run `$ npm start`
+ Open <https://localhost:8080> in a browser.
